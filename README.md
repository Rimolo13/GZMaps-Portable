# GZMaps-Portabel
GZMaps-Portabel is a simple, minimalistic map tool made for my organisation. It is designed to be easy to change, specifically for fast-changing environments, but at the same time to not be prone to accidental changes.

## Technology Stack
GZMaps-Portabel is built with HTML/CSS/JS 
No external libraries are used

## How to Use
Just open the index.html in the src Dir.
