# GZMaps-Portable
GZMaps-Portable is a simple, minimalistic map tool made for my organisation. It is designed to be easy to change, specifically for fast-changing environments, but at the same time to not be prone to accidental changes.

## Technology Stack
GZMaps-Portable is built with HTML/CSS/JS 
No external libraries are used

## How to Use
Use a local host.
Just clicking on the HTML is not possible at the moment.
